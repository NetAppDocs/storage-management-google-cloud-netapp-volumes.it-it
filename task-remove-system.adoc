= Rimuovere i Google Cloud NetApp Volumes dalla NetApp Console
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
È possibile rimuovere un abbonamento a Google Cloud NetApp Volumes e tutti i volumi esistenti dalla NetApp Console.  I volumi non vengono eliminati, vengono rimossi da tutti i progetti all'interno della NetApp Console e possono essere nuovamente rilevati in un secondo momento.


TIP: Non è possibile eliminare Google Cloud NetApp Volumes dalla NetApp Console.  Esegui questa azione utilizzando Google Cloud Console.

.Passi
. Aprire il sistema.
. Fare clic suimage:screenshot_gallery_options.gif["Un'icona con tre punti."] pulsante nella parte superiore della pagina e fare clic su *Rimuovi*.
