[role="lead"]
Per iniziare a utilizzare Google Cloud NetApp Volumes, configura Google Cloud e crea un sistema.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Uno"] link:task-set-up-gcnv.html["Configurare Google Cloud NetApp Volumes"]
[role="quick-margin-para"]
Da Google, crea un account di servizio.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Due"] link:task-create-working-env.html["Aggiungi un sistema"]
[role="quick-margin-para"]
Dalla console NetApp , fare clic su *Aggiungi sistema*, quindi su *Google Cloud Platform*, quindi su * Google Cloud NetApp Volumes* e infine su *Scopri*.  Fornisci i dettagli sull'account di servizio e sul progetto Google Cloud.
